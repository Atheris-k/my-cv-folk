# Curriculum Vitae
## Examen Módulo 1

Este proyecto corresponde al examen final del **Módulo 1: Introducción al Desarrollo Web**, de la carrera **Full Stack JavaScript** de **Desafío Latam (ADL)**, generación **G107**.

## Características
- Estructura semántica en HTML.  
- Estilos básicos en CSS.  
- Ajustes simples para diseño responsivo.
- Manejo de conceptos básicos de la Terminal.
- Conocimientos aplicados de Git.
- Trabajo colaborativo a través de GitHub.
